# 除錯：時間分組測試

## GET: /api/debug/time-grouping
- 測試時間分組功能的正確性API
- 驗證系統在處理時間資料時的分組邏輯是否正常運作。

---


## **請求參數（Request Parameters）**

**Type**：無

此 API 為開發用途，不需要請求參數。

## **回應欄位（Response Fields）**

| Attribute | Type | Required | Description |
|------------|------|-----------|--------------|
| result | object | Y | 時間分組測試結果 |
